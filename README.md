# TicTacToe

A place where you and your friends can play Tic-Tac-Toe and see the history.

## How to run the app

- Clone the project
- Do `git checkout feedbackfruits`

- Inside `frontend`, run the following commands:
  - `npm install`
  - `npm run test`
  - `npm run start`
- Inside `backend`, run the following commands:
  - `bundle install`
  - `rails db:migrate`
  - `rails server`
